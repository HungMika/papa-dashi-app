📦 Dashi Quán App - Hướng dẫn sử dụng

1. Giải nén file DashiApp.zip ra một thư mục bạn muốn (VD: Desktop).
2. Mở thư mục vừa giải nén.
3. Click đúp file: run-app.bat
4. Chờ cho tới khi cả hai cửa sổ hiện lên → App sẵn sàng sử dụng.

📌 Lưu ý:
- Không cần cài gì cả, chỉ cần click.
- App chạy offline.
- Dữ liệu JSON nằm trong app/src/data/
- Ảnh sản phẩm trong app/public/product-img/
- Quy tắc đặc tên ảnh: 
    + Chỉ cần đặc "CB cháo cá hồi.png", ảnh tự động sửa thành "CB-chao-ca-hoi.png"
    + Không cần dùng ký tự đặc biệt như là "CB - cháo cá hồi.png", vì ảnh sẽ thành "CB---chao-ca-hoi.png"
- Nếu muốn xóa dữ liệu, xóa hoặc sửa thủ công, thao tác file trong src/data là đủ.
